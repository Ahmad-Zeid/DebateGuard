# Gemini Live Agent Challenge requirements for DebateGuard

## Category
Live Agents

## Mandatory requirements
- Must be a NEW next-generation AI agent
- Must use multimodal inputs/outputs
- Must move beyond text-in/text-out
- Must use Gemini Live API or ADK
- Must be hosted on Google Cloud
- Must use a Gemini model
- Must be built with Google GenAI SDK or ADK
- Must use at least one Google Cloud service

## Submission requirements
- Public code repository
- README with spin-up instructions
- Text description covering features, technologies, data sources, findings, and learnings
- Proof backend is running on Google Cloud
- Architecture diagram
- Demo video under 4 minutes
- Demo must show real software working in real time, no mockups

## Bonus targets
- Automated cloud deployment via scripts or infrastructure as code
- Optional build writeup or content piece created for this hackathon

## Project-specific interpretation
DebateGuard must demonstrate:
- live, interruptible debate interaction over voice
- webcam-informed coaching
- real-time factual grounding via Google Search
- between-round personalized coaching with a scoring rubric
